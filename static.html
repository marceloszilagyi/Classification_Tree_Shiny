Run a model based on your selection of variables!

  • This Shiny App uses the German Credit Data
  • Data is extracted from the caret package
  • The dataset has more than 60 variables and classifies german bank clients as 'Good' or 'Bad' credit
  • There is a total of 1000 observations and 70% of the observations are classified as 'Good'; 30% are classified as 'Bad'.
  • This shiny app allows you to select the data that will be part of a classification tree

Instructions

  1. There are 3 tabs with data
  2. Select the variables by clicking on the lines in the binary tab
  3. Select the variables using the selection boxes in the categorical and continuous tabs
  4. All tabs display relevant information based on user selection (e.g. box plots for variables
  5. Your selection is shown on the 'Variables Selected' on the left of the screen
  6. Once at least 3 variables are selected, click 'Submit'.
  7. The results will be displayed on the results tab
  8. Please be patient - the model can take some seconds to run - there is code that prevents the use of buttom submit during the calculation
  9. See the results by clicking on the results tab
